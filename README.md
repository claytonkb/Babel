Babel
===========

Tutorial
--------

To run the tutorial in Windows:  
    - Open a command window (Win+R, cmd, OK)  
    - Change to the directory where you cloned the Babel repo  
    - bin\babel.exe  
    - At the prompt, enter: quick\_start !
    - Follow the on-screen directions
